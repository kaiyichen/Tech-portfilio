LinkedOUT is the only application that any experienced recruiter needs. It allows recruiters to keep track of many applicants, and the job they applied for. You can store their contact details, skills and the round of their application, all in one place. LinkedOUT comes with a Command Line Interface CLI as well as a Graphical User Interface GUI in order to provide recruiters a more streamlined experience.

FEEDDO - Feed with Todo , is a mobile application that drives users to maintain a work life balance by introducing pets that will behave according to how close the users are to achieving their target balanced life.

Users will get to set their weekly goals and view their progress in the dashboard page. In order to achieve a higher balance level, users will have to plan their time wisely by dropping down todos according to categories. They will be able to view their daily schedule at the schedule page.

1MBD was created for an orientation and it functions like the monopoly boardgame. Freshies will have to identity the culprit based on a storyline and the boardgame. This boardgame is created using HTML, CSS, Javascript along with a Figma prototype.

DukeTime is a todo app to allow users to keep track of their todos, deadlines and upcoming events. This is a project template for a greenfield Java project. It's named after the Java mascot Duke.
